Rhfiles\API\User::login_get

This function is called when a get request is sent to the api controller with parameters of login, The api uses the user model to check for a matching $email and $enc_pass.If successful will return a JSON object and HTTP_OK (200) response code or HTTP 404 response code A $email and $enc_pass value must be provided in the get request
