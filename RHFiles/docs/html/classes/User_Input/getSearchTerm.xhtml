User_Input::getSearchTerm

This function is used to return the search term that has been POST'ed when searching the skill table.

If the search term is not stored in a session it will be forgotten so this method stores the term as session data if one does not already exist. This is so the next SELECT LIMIT WHERE query can be ran with this same search term. Returning every record that matches the search term snd storing it would not scale compared to carrying out a SELECT WHERE LIMT query and using pagination to increment the offset of limit.
