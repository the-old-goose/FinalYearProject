Gravatar::get

Creates a URL for requesting a Gravatar image.

    Signature

    public function get(string $email, [int $size = NULL, [string $default_image = NULL, [bool $force_default_image = NULL, [string $rating = NULL]]]] )

    Parameters

    $email — string
    A registered email.
    $size — int
    The requested size of the avarar in pixels (a square image).
    $default_image — string
    The fallback image option: '', '404', 'mm', 'identicon', 'monsterid', 'wavatar', 'retro', 'blank'.
    $force_default_image — object
    Enforces the fallback image to be shown.


    $rating — string
    The level of allowed self-rate of the avatar: '', 'g' (default), 'pg', 'r', 'x'.

    Returns

    string
    Returns the URL of the avatar to be requested.
    When optional parameters are not set, their default values are taken from the configuration file application/config/gravatar.php
