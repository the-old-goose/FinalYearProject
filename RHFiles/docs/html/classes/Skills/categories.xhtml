Skills::categories

This function is called when the user clicks a category on the marketplace page(skill index view).

The categories id is passed in the URI and HTTP GET request to the api returns all the skills which have a corrisponding category_id as a JSON which is converted to an associative array and passed onto the view.

    Signature

    public function categories([integer $c_id = 0, [integer $offset = 0]] )

    Parameters

    $c_id — integer
    Id of the category to get skills for.
    $offset — integer
    Incremented by pagination, used to calculate which records to return from DB
