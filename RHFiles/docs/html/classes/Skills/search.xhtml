Skills::search

This function is called when the user searches for a term on the skills view.A http POST request contains the search term. The search term is set by the @method getSearchTerm().

A JSON file is returned after a HTTP GET request is sent to the api with the searchterm , offset and results per page.The JSON is decoded into an associative array which is passed to the view.
