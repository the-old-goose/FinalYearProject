Rhfiles\API\Skill::incategory_get

This function is called when a get request is sent to this api controller with parameters of incategory, The api uses the skill model to retrieve specific skills from the DB if successful will return a JSON object and HTTP_OK (200) response code or HTTP 404 response code A $c_id, $offset and $items integer value must be provided in the get request
